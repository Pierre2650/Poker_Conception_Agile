Member Function Documentation

◆ checkUnanimity()

void Results_Review_Next_Controller.checkUnanimity ( )

Methode qui verifie l'unanimite des votes et effectue l'action appropriee.

Cette methode verifie Si l'unanimite est atteinte (controller.unanimity est vrai), si la valeur indisputee est "Coffee", elle lance une pause cafe en appelant coffeeBreak. Sinon, elle termine le match en appelant endMatch, si l'unanimite n'est pas atteinte, elle lance un debat en appelant startDebate.

◆ nextbutton()

void Results_Review_Next_Controller.nextbutton ( )

Methode appelee lors de l'appui sur le bouton "Next".

Cette methode verifie Si le mode de jeu est "Average" et que le round actuel est le deuxieme, la methode termine le match en appelant endMatch. Sinon, elle verifie l'unanimite des votes en appelant checkUnanimity, Dans les autres modes de jeu, elle appelle directement checkUnanimity.
